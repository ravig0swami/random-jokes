# Random Joke Generator

Random Joke Generator 🎭 – A lightweight and responsive web app built with React, Vite, and Tailwind CSS that fetches and displays random jokes from the Official Joke API. Features a clean UI, setup &amp; punchline delivery, and fast performance. Perfect for learning React, working with APIs, and building fun frontend projects.

## Features

- **Get Random Jokes:** Click a button to fetch a new random joke on demand.
- **Clean & Responsive UI:** A simple, modern, and responsive user interface that looks great on desktops, tablets, and mobile devices.
- **Setup & Punchline:** Displays both the setup and the punchline for a complete joke experience.
- **Built with Modern Tools:** Utilizes React for the UI, Vite for a blazing-fast development experience, and Tailwind CSS for utility-first styling.

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Make sure you have the following installed on your machine:
- [Node.js](https://nodejs.org/en/) (v18.x or higher is recommended)
- [npm](https://www.npmjs.com/) (comes with Node.js)

### Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/ravig0swami/random-jokes.git
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd random-jokes
    ```

3.  **Install dependencies using npm:**
    ```bash
    npm install
    ```
4.  **Run the development server:**
    ```bash
    npm run dev
    ```
    The application will be available at `http://localhost:5173` (or another port if 5173 is busy).

5.  **To create a production-ready build, run:**
    ```bash
    npm run build
    ```
    This will create a `dist` folder with the optimized static assets.
